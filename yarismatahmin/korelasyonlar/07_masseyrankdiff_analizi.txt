## DEĞİŞKEN TANIMI

**MasseyRankDiff:** Kazanan takımın Massey rank'i ile kaybeden takımın Massey rank'i arasındaki fark

**Formül:**
```
MasseyRankDiff = W_MasseyRank - L_MasseyRank
```

**Massey Ranking Sistemi:**
- **Düşük rank = Daha iyi takım** (1 = en iyi, 350+ = en kötü)
- 196 farklı Massey sıralama sistemi var
- Son RankingDayNum'daki ortalama rank kullanılıyor

---

## ÖNEMLİ NOT: TERS MANTIK!

**Dikkat:** Massey rank'te düşük sayı = iyi takım

**Örnek:**
- Duke: Rank 5 (çok iyi)
- FGCU: Rank 200 (zayıf)
- RankDiff = 5 - 200 = **-195** (negatif = Duke daha iyi)

**SeedDiff ile Karşılaştırma:**
- SeedDiff: Negatif = daha iyi seed kazandı
- MasseyRankDiff: Negatif = daha iyi rank kazandı
- **İkisi de aynı mantık!**

## MODEL İÇİN KULLANIM

### 1. Feature Importance

**MasseyRankDiff:** Çok önemli feature
- Korelasyon: ~-0.75
- Tek başına %69.8 accuracy
- **2. en güçlü predictor**

**Kullanım:**
```python
# Direkt kullan
features = ['SeedDiff', 'MasseyRankDiff', ...]

# Normalize et (0-1 arası)
MasseyRankDiff_Normalized = (MasseyRankDiff - min) / (max - min)
```

### 2. Interaction Features

**SeedDiff × MasseyRankDiff:**
```python
Seed_Massey_Interaction = SeedDiff * MasseyRankDiff
```

**Anlaşma Feature:**
```python
if (SeedDiff < 0 and MasseyRankDiff < 0) or (SeedDiff > 0 and MasseyRankDiff > 0):
    Agreement = 1  # İki sistem aynı fikirde
else:
    Agreement = 0  # İki sistem farklı fikirde
```

### 3. Weighted Average

**SeedDiff ve MasseyRankDiff'i birleştir:**
```python
Combined_Rank = 0.5 * SeedDiff_Normalized + 0.5 * MasseyRankDiff_Normalized
```
