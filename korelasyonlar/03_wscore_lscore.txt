## MODEL İÇİN KULLANIM

### 1. PointDiff Feature
**Regular Season PointDiff:** Takımın güç göstergesi
```python
Team_Avg_PointDiff = (WScore - LScore).mean()
```

### 2. Score Variance
**Skor Dalgalanması:** Tutarsız takımlar
```python
Score_Variance = (WScore - LScore).std()
```

### 3. High/Low Scoring Team
**Yüksek skorlu takım:** Ofansif güç
```python
Is_High_Scoring = WScore.mean() > 80
```
